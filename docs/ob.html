ob.deselect(subject, keys){object}

Returns an object without the given keys.

Arguments

subject object | Array.<object>
keys Array.<string>
Returns:
Type Description
object
Examples

Basic usage.


let x = {
 a: 1,
 b: 2,
 c: 3,
}
ob.deselect(x, [a,b]);
// Returns {c: 3}

Advanced usage.


let x = {
 a: 1,
 b: 2,
 c: 3,
 d: {e: 4, f: [5,6]},
 g: [7, 8]
}
