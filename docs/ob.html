ob.deselect(subject, keys){object|Array.<object>}

Returns an object without the given keys or an array with each object not having the given keys.

Arguments

  1. subject ( object | Array.<object> ) :

    The object or array to perform the deselect operation on.
  2. keys ( Array.<string> ) :

    The keys of the object or nested object that you would like to deselect.
Returns:
( object | Array.<object> ) : The object or array of objects without the deselected keys
Examples

Basic usage.


let x = {
 a: 1,
 b: 2,
 c: 3,
}


ob.deselect(x, ['a','b']);
// → {c: 3}

Advanced usage.


let x = {
 c: 3,
 d: {e: 4, f: [5,6]},
 g: [7, 8]
}


ob.deselect(x, ['d.e','d.f[].0','g[].1']);
// → {c: 3, d: {f:[6]}, g:[7]}

ob.expand(subject){object|Array.<object>}

    The object to expand
Returns:

Example
let x = {

 'a.b.d': [2,3]
 'a.b.d[].0': 2,

ob.select(subject, keys){object|Array.<object>}

Returns an object only with the given keys. If an array is passed, it will return an array of each given object only having the selected keys.

Arguments

  1. subject ( object | Array.<object> ) :

    The object or array of objects to perform the select operation on
  2. keys ( Array.<string> ) :

    The keys you would like to select
Returns:
( object | Array.<object> ) : The object or array of objects with only the selected keys.
Examples

Basic usage.


let x = {
 a: 1,
 b: 2,
 c: 3,
}


ob.select(x, ['a','b']);
// → {a: 1, b: 2}

Advanced usage.


let x = {
 c: 3,
 d: {e: 4, f: [5,6]},
 g: [7, 8]
}


ob.select(x, ['d.e','d.f[].0','g[].1']);
// → {d: {e: 4, f: [5]}, g: [8]}
