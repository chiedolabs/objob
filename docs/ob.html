ob.deselect(subject, keys){object}

Returns an object without the given keys.

Arguments

  1. subject ( object | Array.<object> ) :

    The array or object to perform the deselect operation on.
  2. keys ( Array.<string> ) :

    The keys of the object or nested object that you would like to deselect.
Returns:
( object ) : The object without the deselected keys
Examples

Basic usage.


let x = {
 a: 1,
 b: 2,
 c: 3,
}


ob.deselect(x, ['a','b']);
// → {c: 3}

Advanced usage.


let x = {
 c: 3,
 d: {e: 4, f: [5,6]},
 g: [7, 8]
}


ob.deselect(x, ['d.e','d.f[].0','g[].1']);
// → {c: 3, d: {f:[6]}, g:[7]}

ob.expand(subject, depth){object|Array.<object>}

Takes a flattened object or array and expands it back to a full object
or array.

Arguments

  1. subject ( object | Array.<object> )
  2. depth ( integer default=1 ) optional
Returns:
( object | Array.<object> )
Example
let x = {
 'a.b.c': 1,
 'a.b.d': [2,3]
 'a.b.d[].0': 2,
 'a.b.d[].1: 3',
}


ob.expand(x)
// → {a: {b: {c: 1, d: [2,3]}}}
